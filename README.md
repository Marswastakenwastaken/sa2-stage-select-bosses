# SA2 Stage Select Bosses

## Description

This mod adds a new Stage entry on the Stage Selection Map. The new "stage" will appear above Green Hill. (To the left of Aquatic Mine)
Clicking this entry will bring up a selection for every boss fight available in all Story Modes as single level entries.

## Getting Started

### Dependencies

* Windows
  * Linux running Wine should also work but this tool is not designed for or tested under Linux.
* [Sonic Adventure 2 - Steam Version](https://store.steampowered.com/app/213610/Sonic_Adventure_2/)
* [SA Mod Manager](https://github.com/X-Hax/SA-Mod-Manager/releases/latest)
	* And all of its dependencies

### Installing

The latest version can be found for download as on the [releases tab.](https://github.com/QueenSuzie/sa2-stage-select-bosses/releases)

Alternatively, a one click install option is available on [Game Banana].(https://gamebanana.com/mods/640918)
